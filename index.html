Embedded Chat

Experience Savi as a fully integrated chat interface on your page. Perfect for dedicated support pages or when you want the chat to be the main focus.

Implementation code:

<div id="customgpt_chat"></div>
<script src="https://cdn.customgpt.ai/js/embed.js" div_id="customgpt_chat" p_id="82753" p_key="ea4e59faa461427ca9e80161e2de77c0"></script>

Live Chat Widget

Add a floating chat widget to any page. Users can access Savi support without leaving their current page - perfect for e-commerce and service websites.

Implementation code:

<script defer src="https://cdn.customgpt.ai/js/chat.js"></script>
<script defer>(function(){function init(){CustomGPT.init({p_id:'82753',p_key:'ea4e59faa461427ca9e80161e2de77c0'})}document.readyState==='complete'?init():window.addEventListener('load',init);})();</script>

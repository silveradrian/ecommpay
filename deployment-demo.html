Savi Deployment Options

Choose how to integrate Savi into your website

Embedded Chat

Embed the Savi chatbot directly into your website with full customization control. Perfect for dedicated support pages or when you want the chat to be the main focus.

Implementation:

<!-- Add a div where you want the chat to appear --> <div id="customgpt_chat"></div> <!-- Add the embed script at the end of body --> <script src="https://cdn.customgpt.ai/js/embed.js" defer div_id="customgpt_chat" p_id="82753" p_key="ea4e59faa461427ca9e80161e2de77c0"> </script>

Live Chat Widget

Add a floating chat widget to any page on your website. Users can access Savi support without leaving their current page - perfect for e-commerce and service websites.

Implementation:

<!-- Add this before the closing </body> tag --> <script defer src="https://cdn.customgpt.ai/js/chat.js"></script> <script defer> (function(){ function init(){ CustomGPT.init({ p_id:'82753', p_key:'ea4e59faa461427ca9e80161e2de77c0' }) } document.readyState === 'complete' ? init() : window.addEventListener('load', init); })(); </script>

Which Option is Right for You?

Choose Embedded Chat if:

  • • You have a dedicated support or contact page
  • • You want the chat to be the primary focus
  • • You need full control over the chat dimensions
  • • You prefer a seamless, integrated experience

Choose Live Chat Widget if:

  • • You want support available on every page
  • • You prefer a non-intrusive solution
  • • Your users need help while browsing
  • • You want quick, easy implementation
